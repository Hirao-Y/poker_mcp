# 対象視聴者分析

## 1. 視聴者セグメント

### セグメント1: 新規研究者 (30%)
**基本情報**
- 経験年数: 0-2年
- 教育背景: 物理学、原子力工学、放射線工学の学士・修士
- 技術レベル: 理論的知識はあるが実践経験が限定的

**特徴とニーズ**
- 放射線遮蔽の基礎概念を実践に応用したい
- 直感的で分かりやすい説明を求める
- 段階的な学習プロセスを重視
- エラーを恐れずに試行錯誤できる環境が必要

**視聴傾向**
- 基礎から丁寧に学びたい
- 実演を見ながら同時に操作したい
- 繰り返し視聴する傾向
- コメント機能でよく質問する

### セグメント2: 経験豊富な技術者 (40%)
**基本情報**
- 経験年数: 3-10年
- 職場: 原子力施設、研究機関、エンジニアリング会社
- 技術レベル: 他のツールでの計算経験豊富

**特徴とニーズ**
- 効率的なワークフローを求める
- 既存知識との関連付けを重視
- 時間を節約したい（要点を素早く把握）
- 高度な機能への関心が高い

**視聴傾向**
- 目的の機能に直接アクセス
- 倍速再生を多用
- 具体的な数値例を重視
- 他ツールとの比較に興味

### セグメント3: 研究指導者・管理者 (20%)
**基本情報**
- 経験年数: 10年以上
- 役職: プロジェクトリーダー、研究室主任、部門管理者
- 責任: チームの技術レベル向上、プロジェクト管理

**特徴とニーズ**
- チーム全体の教育効果を重視
- 品質保証と標準化に関心
- ROI（投資対効果）を評価したい
- ベストプラクティスの確立を求める

**視聴傾向**
- 概要と総合的な評価に注目
- 部下への推奨判断材料として利用
- ケーススタディに高い関心
- コスト・効率性の情報を重視

### セグメント4: 学生・教育関係者 (10%)
**基本情報**
- 立場: 大学院生、ポスドク、大学教員
- 目的: 研究、教育、論文作成
- 制約: 限られた時間とリソース

**特徴とニーズ**
- 理論と実践の橋渡し
- 教育材料としての活用
- 研究の再現性確保
- アカデミックな正確性重視

**視聴傾向**
- 理論的背景の説明を重視
- 参考文献や根拠を求める
- 教育利用の許可条件に注意
- 研究事例の詳細に関心

## 2. 共通特性分析

### 技術的背景
**強み**
- 放射線物理学の基礎知識
- 数学的思考力
- 科学的アプローチの習慣
- 専門ソフトウェアの使用経験

**弱み・課題**
- 3D幾何学的思考の不慣れ
- プログラミング経験の個人差
- 新しいツールへの適応時間
- 操作ミスによるデータ損失の恐れ

### 学習スタイル
**視覚型学習者 (60%)**
- 図解、動画、実演を好む
- 色分けや視覚的階層化が効果的
- スクリーンショットや図表を重視

**聴覚型学習者 (25%)**
- 音声説明を重視
- 理論的説明を好む
- ディスカッションや質疑応答を活用

**体験型学習者 (15%)**
- 実際の操作体験を重視
- 試行錯誤を通じた学習を好む
- ハンズオン形式を求める

### 動機とゴール
**短期目標**
- 基本操作の習得
- 実務での即戦力化
- エラー削減

**中期目標**
- 効率的なワークフロー確立
- 高度な機能の活用
- 品質向上

**長期目標**
- 専門性の向上
- チーム・組織での標準化
- 研究・業務の革新

## 3. 視聴環境と制約

### 技術環境
**デバイス**
- デスクトップPC (70%)
- ノートPC (25%)
- タブレット (5%)

**ネットワーク**
- 高速インターネット (組織内)
- 帯域制限あり (一部在宅勤務)
- セキュリティ制約 (ダウンロード制限)

### 時間的制約
**平日の視聴パターン**
- 業務時間: 30-45分の集中視聴
- 昼休み: 15-20分の短編視聴
- 残業時間: 疲労を考慮した内容

**週末・休日**
- 自宅学習: 60-90分の集中学習
- 復習・確認: 特定セクションの再視聴

### 組織的制約
**承認・予算**
- 上司の承認が必要な場合
- 研修予算の制限
- 時間確保の困難

**セキュリティ**
- 外部動画サービスの制限
- データ持ち出し制限
- 録画・共有の制約

## 4. ニーズ別コンテンツ要求

### 内容の深さ
**基礎レベル**
- 概念の丁寧な説明
- 前提知識の確認
- 段階的な導入

**応用レベル**
- 実践的なテクニック
- 効率化の方法
- トラブルシューティング

**専門レベル**
- 高度な機能活用
- カスタマイゼーション
- 最適化手法

### 説明スタイル
**理論重視型**
- 原理・メカニズムの説明
- 根拠となる理論の提示
- 学術的な正確性

**実践重視型**
- 手順の明確な提示
- 実例を用いた説明
- すぐに使える情報

**問題解決型**
- よくある問題の対処法
- エラー事例と解決策
- 予防方法の提示

## 5. 成功要因

### エンゲージメント向上
**内容の質**
- 正確で最新の情報
- 実務に直結する内容
- 段階的な難易度設定

**表現方法**
- 明瞭な音声と映像
- 適切なペースの説明
- 視覚的にわかりやすい表示

**インタラクティブ要素**
- 練習問題の提供
- チェックポイントの設置
- 質疑応答の機会

### 継続的な改善
**フィードバック収集**
- 定期的なアンケート
- 使用状況の分析
- 個別ヒアリング

**コンテンツ更新**
- 新機能への対応
- ユーザー要望の反映
- 最新事例の追加

## 6. 測定指標

### 視聴指標
- 視聴開始率
- 完了率
- 再視聴率
- セクション別離脱率

### 学習効果指標
- 理解度テスト結果
- 実務適用率
- エラー発生率の変化
- 作業時間短縮効果

### 満足度指標
- ユーザー評価
- 推奨度 (NPS)
- フィードバック内容
- 継続利用率

- 複数の計算ツールを使いこなす
- 効率重視、新しいツールに積極的

### ペルソナ3: 山田教授 (研究指導者)
