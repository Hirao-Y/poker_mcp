# 🎥 Poker MCP 動画ガイド制作計画書

## 📋 制作スケジュール

### Phase 1: 基本動画（2-3週間）
- **Week 1**: セットアップ + 基本線源操作
- **Week 2**: 簡単実験ワークフロー + 立体管理基礎
- **Week 3**: 編集・公開準備

### Phase 2: 応用動画（3-4週間）
- **Week 4-5**: トラブルシューティング + バックアップ
- **Week 6-7**: 高度機能 + 最終編集

## 🎬 各動画の詳細企画

### 1. セットアップガイド（15分）
**ターゲット**: 完全初心者
**構成**:
- 0:00-2:00 システム要件確認
- 2:00-5:00 インストール実演
- 5:00-10:00 初期設定
- 10:00-13:00 動作確認
- 13:00-15:00 よくある問題と解決

**準備物**:
- クリーンなWindows環境
- 事前インストールしたDocker
- 準備済みスクリプト

### 2. 基本線源操作（7分）
**ターゲット**: 初心者研究者
**構成**:
- 0:00-1:00 APIの基本概念
- 1:00-3:00 線源作成実演
- 3:00-5:00 線源更新・削除
- 5:00-6:30 結果確認
- 6:30-7:00 安全性ポイント

**準備物**:
- 動作済みPoker MCPシステム
- 予め準備したAPIキー
- 実演用JSONデータ

### 3. 実験ワークフロー（10分）
**ターゲット**: 研究実施者
**構成**:
- 0:00-1:30 実験目的説明
- 1:30-4:00 自動化スクリプト実行
- 4:00-7:00 結果の解釈
- 7:00-9:00 応用例
- 9:00-10:00 次のステップ

**準備物**:
- 研究ワークフロースクリプト
- サンプルデータセット
- 結果表示システム
